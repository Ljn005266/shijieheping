一、项目简介
本仓库为BNGU视觉组招新考核的计算机视觉基础任务实现，包含以下实验：
1.  OpenCV 环境配置与测试
2.  颜色分割实验
3.  多边形检测与最小框选实验
4.  Tag识别与位姿解算实验

所有代码基于Python + OpenCV实现，可直接运行。

---

二、环境配置
 1. 依赖安装
在终端执行以下命令安装依赖：
# 创建虚拟环境
python -m venv .venv
# 激活虚拟环境（Windows）
.venv\Scripts\activate
# 安装OpenCV
pip install opencv-python opencv-contrib-python -i https://pypi.tuna.tsinghua.edu.cn/simple
